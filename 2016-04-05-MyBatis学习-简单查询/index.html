MyBatis学习-简单查询

发表于 2016-04-05 | 分类于 mybatis | 阅读次数:
字数统计: 894 | 阅读时长 ≈ 5

查询分为返回单个结果和返回一个结果集。而结果集分为List，Map，其中List，且两者都是泛型，根据映射文件中配置的reusltMap，mybatis会对查询的结果进行封装
概述

通过SqlSession可以看出，查询可分为以下几组：

  • T selectOne(…):返回给定类型的单个结果集
  • List selectList(…):返回给定类型的多个结果集
  • Map selectMap(…):返回指定列为key的map集合或对象
  • void select(…) : 检索来自该语句映射的一个单列

返回单条结果集

  • T selectOne(String statement);
  • T selectOne(String statement,Object paramter);

这两组是选择单个结果集。如果后台是返回多个结果却用这个方法来进行接收的话，则会报错：多个结果集无法匹配单个结果集。

返回列表结果集

  • List selectList(String statement) ;
  • List selectList(String statement,Object paramer) ;
  • List selectList(String statement, Object paramter,RowBounds rowBounds);

返回Map结果集

  • Map selectMap(Strng statement, String mapKey);
  • Map selectMap(String statement,Object paramter,String mapKey);
  • Map selectMap(String statement, Object parameter, String mapKey, RowBounds rowBounds);

无返回值结果集

  • void select(String statement, ResultHandler handler);
  • void select(String statement, Object parameter, ResultHandler handler);
  • void select(String statement, Object parameter, RowBounds rowBounds, ResultHandler handler);

这种方式用得比较少，就不演示后面两个了

注意事项

  1. 如果后台返回多个结果，而前台是使用selectOne来接受的话，会报错。
  2. 关于resultType，写不写好像没啥关系
